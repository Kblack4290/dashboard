About Financial Dashboard

Application Overview

The Financial Dashboard is a comprehensive tool designed to help users monitor and analyze stock market data. This application provides real-time stock information, historical price charts, and personalized watchlists to help users make informed investment decisions.

Key Features

  • Stock Data Visualization: View interactive charts of stock price history with 7-day and 30-day views
  • Stock Search: Look up any stock by its symbol to get the latest price data and performance metrics
  • Personalized Watchlist: Save your favorite stocks to a watchlist for quick access and monitoring
  • Automated Data Updates: Scheduled service fetches the latest market data to keep information current

Technology Stack

Frontend

  • Framework: Angular (Standalone Components)
  • Charting: ng2-charts (Chart.js)
  • Styling: CSS with responsive design

Backend

  • Framework: ASP.NET Core
  • Database: PostgreSQL with Entity Framework Core and migrations
  • API Integration: Alpha Vantage API for stock market data
  • Background Services: Scheduled data fetching with .NET background services

How to Use

  1. View Stock Data: Enter a stock symbol (e.g., AAPL, MSFT, GOOG) in the search box and click "Search" to view the latest data and price charts.
  2. Toggle Time Range: Switch between 7-day and 30-day views to analyze short and medium-term trends.
  3. Manage Your Watchlist: Add stocks to your watchlist by clicking the "Add to Watchlist" button while viewing a stock. Remove them by clicking the "Remove from Watchlist" button.
  4. Access Your Watchlist: Navigate to the Watchlist page to see all your saved stocks in one place with their latest performance metrics.

Data Source

This application uses stock market data provided by the Alpha Vantage API. The data is cached in our PostgreSQL database and updated regularly to ensure accuracy while minimizing external API calls.

Note: Stock data may be delayed by up to 24 hours depending on the last database update.
